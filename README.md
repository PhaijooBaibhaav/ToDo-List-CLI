# ToDo List CLI made with **C#**
## Description
### This is a ToDO list CLI made with C#. With this we can add tasks, delete specific tasks or clear all tasks.

## Requirements
- Git
- Dotnet SDK

## Installation
```bash
git clone https://github.com/PhaijooBaibhaav/ToDo-List-CLI
cd ToDo-List-CLI
dotnet run
```
